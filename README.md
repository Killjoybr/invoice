# Invoice

## Desenvolvimento

Este projeto está sendo desenvolvido utilizando o XAMPP, então esta seção foca no funcionamento do formulário por ele.

Clone o repositório na pasta `xampp/htdocs`

Inicie o servidor apache na interface do xampp, e após isso você poderá acessar via `http://localhost:PORTA_CONFIGURADA/invoice`

Note que a porta configurada geralmente é a 80 ou 8080, troque [PORTA_CONFIGURADA] pela porta que você configurou.
